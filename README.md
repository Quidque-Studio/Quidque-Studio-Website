# Quidque Studio Website

Portfolio and blog site for Quidque Studio. Features projects with devlogs, team member pages, studio blog, and newsletter.

## Routes

| Base Path   | What it covers                                    |
|-------------|---------------------------------------------------|
| `/`         | Home, about, search                               |
| `/projects` | Projects, devlogs                                 |
| `/blog`     | Studio posts                                      |
| `/team`     | Team member profiles and personal posts           |
| `/auth`     | Magic link login, registration                    |
| `/messages` | User-to-admin messaging                           |
| `/admin`    | Project, post, user, newsletter management        |

## Tech

- Vanilla PHP, CSS & JS
- MariaDB
